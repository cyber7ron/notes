if,else,elif statement

If,else , elif statement

Conditional statement allow use to perform different action on different conditions..
the if statement checks if a condition is true based on this it executes some cmds..

syntax

if [ <some condition> ]
then
    //some statement here
fi

if else

if else allow two code execution branches.

syntax

if [ <some test> ]
then
    //perform some action
else
    //perform another action
fi

also we can extend functionality using elif statement.

if [ <some condition> ]
then
    //perform this action
elif [ <some test> ]
    //perform this action
else
    //perform this action
fi
